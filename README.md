# diet-app
# 🥗💰 [프로젝트 이름] : 다이어트 & 재테크 결합 앱

## 1. 🌟 프로젝트 개요 (Project Overview)

본 프로젝트는 **건강 관리**와 **재테크** 개념을 결합한 모바일 애플리케이션입니다. 사용자가 다이어트 습관을 기록하고 목표를 달성할 때마다 **포인트로 보상**받아 이를 실제 리워드(상품, 쿠폰, 기프티콘)로 교환할 수 있는 동기 부여 시스템을 제공합니다. 사용자의 꾸준한 건강 관리 습관 형성을 목표로 합니다.

## 2. 🔑 핵심 구현 기능 (Core Features)

| 카테고리 | 기능 | 상세 설명 | 포인트 지급 여부 |
| :--- | :--- | :--- | :--- |
| **활동 기록** | 출석체크 | 매일 앱 로그인 시 출석 기록 및 포인트 지급 | ⭕️ |
| **활동 기록** | 운동 기록 | 사용자가 **운동 종류, 시간, 소모 칼로리**를 직접 기록 및 관리 | ⭕️ |
| **활동 기록** | 몸무게 기록 | 사용자가 **현재 몸무게를 직접 입력**하여 기록 및 관리 | ⭕️ |
| **식단 관리** | 식단 기록 | 사용자가 **섭취한 음식의 이름과 양**을 직접 입력하여 기록 | ⭕️ |
| **식단 관리** | 영양소 시각화 | 기록된 식단 정보를 기반으로 **탄수화물 / 단백질 / 지방** 비율을 원그래프로 시각화하여 제공 | ❌ |
| **포인트 시스템**| 포인트 지급 | **출석, 운동 기록, 몸무게 기록, 식단 기록, 광고 노출/클릭** 등 활동 시 포인트 지급 | ⭕️ |
| **포인트 시스템**| 리워드 교환 | 적립된 포인트를 사용하여 **상품, 쿠폰, 기프티콘** 등 실제 리워드로 교환 가능 | ❌ |

## 3. 🚫 미구현 및 변경된 기능 (Scope Exclusions)

프로젝트의 초기 기획 중 다음 기능들은 구현 범위에서 제외되었거나 간소화되었습니다.

* **AI 기반 기능 제외:** 식단 기록 시 AI 분석 및 맞춤 영양 추천 기능은 제외되었습니다.
* **시각화 제외:** 기록된 몸무게의 히스토리를 캘린더나 그래프로 표시하는 기능은 제외되었습니다.
* **커뮤니티 기능 제외:** 사용자 간 게시판, 정보 공유, 팔로우 등 모든 커뮤니티 기능은 제외되었습니다.
* **외부 연동/검증 제외:**
    * 걸음수 연동 (Google Fit/HealthKit) 기능은 제외되었습니다.
    * 몸무게 사진 업로드 및 AI 검증 기능은 제외하고, **사용자 직접 입력** 방식으로 간소화되었습니다.
